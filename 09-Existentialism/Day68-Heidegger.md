# Day 68｜海德格爾爾：從意識轉向存在

## 今日主題

```text
Day 68｜海德格爾：從意識轉向存在
Martin Heidegger
```

今天，我們來到 20 世紀哲學最重要的轉折點之一。  
如果說胡塞爾（Edmund Husserl）問的是：

```text
意識如何經驗世界？
```

那麼海德格爾（Martin Heidegger）會說：

```text
你問得還不夠根本。
```

真正的問題不是：

- 我們如何認識世界？
- 意識如何構成對象？
- 真理如何在經驗中成立？

而是：

```text
存在（Being / Sein）到底是什麼？
```

---

## 一、哲學遺忘了一個最根本的問題

海德格爾認為，  
從古希臘到近代哲學，  
人們一直忙著研究：

- 物體
- 真理
- 認識
- 科學
- 上帝
- 心靈
- 主體

但卻忽略了一個更根本的問題：

```text
存在本身
```

他把這稱為：

```text
存在問題（Question of Being / Seinsfrage）
```

這不是普通抽象問題。  
在海德格爾看來，  
如果我們連“存在”是什麼都沒有真正問清楚，  
那麼其他哲學問題其實都已經默默預設了某種存在理解，  
卻沒有自覺。

所以哲學最大的問題不是答案錯了，  
而是：

```text
最根本的問題被忘了
```

---

## 二、什麼叫做「存在」？

先看幾個簡單句子：

- 一棵樹存在
- 一張桌子存在
- 一個人存在
- 一部電腦存在

日常語言裡，  
我們很自然會說這些東西“存在”。  
但哲學家通常會接著問：

- 樹是什麼？
- 人是什麼？
- 桌子的本質是什麼？

海德格爾卻要再往前追問一步：

```text
這些東西為什麼能夠作為「存在的東西」被理解？
```

也就是說，  
他不是先問“某個東西的特性”，  
而是問：

```text
我們所說的「是」「存在」，
究竟意味著什麼？
```

這就是海德格爾真正的問題方向。

---

## 三、存在者與存在

海德格爾最重要的區分之一就是：

```text
存在者（beings / entities / das Seiende）
vs
存在（Being / Sein）
```

### 1. 存在者

存在者就是具體存在的東西。  
例如：

- 樹
- 石頭
- 人
- 動物
- 電腦
- 椅子

它們是我們在世界裡會碰到的各種“東西”。

### 2. 存在

存在不是另一種更大的東西，  
不是藏在萬物背後的一個神祕實體。  
更準確地說，  
它指的是：

```text
存在者如何作為存在者而被理解、被顯明、被開啟
```

所以可以先用入門方式記成：

```text
存在者 = 東西
存在 = 東西之所以作為存在者而顯現的方式或意義
```

這個區分後來常被稱作：

```text
存在論差異（ontological difference）
```

也就是海德格爾整個問題意識的起點。

---

## 四、西方哲學的遺忘

海德格爾認為，  
從柏拉圖（Plato）之後，  
西方哲學逐漸形成一種傾向：

```text
只關心存在者，
忘記存在本身
```

例如哲學常常問：

- 什麼是真正存在的東西？
- 物質與形式哪個更根本？
- 主體如何認識客體？
- 哪種實體最真實？

這些問題都在談“什麼東西存在、怎麼存在”，  
卻很少真正追問：

```text
存在本身的意義是什麼？
```

海德格爾把這種歷史狀態稱作：

```text
存在的遺忘（forgetfulness of Being）
```

這不是說前人完全沒提過存在。  
而是說存在一直被當成理所當然的背景詞，  
卻沒有被徹底追問。

---

## 五、胡塞爾的不足

海德格爾曾是胡塞爾的學生，  
而且《存在與時間》（Being and Time / Sein und Zeit）  
最初還題獻給胡塞爾。  
但他認為胡塞爾的現象學仍然有局限。

問題在於：

```text
胡塞爾仍然把哲學起點放在意識
```

也就是：

- 意識如何指向對象
- 對象如何在意識中被構成
- 真理如何在經驗中成立

海德格爾則認為，  
這樣仍然太像近代哲學的延長。  
因為人在成為“觀察對象的意識”之前，  
其實早就已經：

```text
存在於世界之中
```

所以真正起點不是意識，  
而是：

```text
人的存在方式
```

這就是他對胡塞爾最重要的改造。

---

## 六、人不是旁觀者

傳統哲學很容易預設一個圖式：

```text
主體
↓
觀察世界
```

好像人先是一個獨立內在主體，  
然後再去面對外部世界。  
海德格爾認為這個圖式根本就有問題。

實際情況不是：

```text
先有我，
再有世界
```

而是：

```text
我從一開始就已經在世界裡
```

人不是站在世界外面觀察它的眼睛。  
人也不是先封閉在心靈裡，  
再透過認識活動連到世界。  
相反地：

```text
人原本就是與世界糾纏在一起的存在
```

這一點非常關鍵。  
因為一旦這樣理解，  
哲學的出發點就會整個改變。

---

## 七、在世存在（Being-in-the-world）

海德格爾最著名的核心概念之一就是：

```text
Being-in-the-world
在世存在（In-der-Welt-sein）
```

這個詞不是說  
“人待在世界這個容器裡”。  
它真正的意思是：

```text
人與世界本來就不可分離
```

也就是說，  
人的存在不是孤立的，  
而總是已經在某種情境、實踐、關聯與意義網絡中展開。

例如你看到咖啡杯，  
通常不是先看到一個中性物體，  
再加上意義。  
你多半是直接知道：

```text
那是拿來喝咖啡的
```

這說明我們與世界的關係  
首先不是理論觀看，  
而是實際投入。  
所以“在世存在”就是要打破  
主體 vs 客體的傳統分裂。

---

## 八、工具的例子：槌子（hammer）

海德格爾最經典的例子之一就是：

```text
槌子（hammer）
```

傳統哲學可能會問：

- 槌子的物質是什麼？
- 槌子的形狀如何？
- 槌子的性質如何被認識？

但海德格爾會說：  
人最先不是“觀察槌子”，  
而是：

```text
使用槌子
```

也就是說，  
槌子首先不是作為“客觀物體”出現，  
而是作為：

```text
可用來敲打的東西
```

在此，世界首先呈現為：

- 可用的
- 有關聯的
- 有目的網絡的
- 與實踐相連的

因此海德格爾的重要結論是：

```text
世界首先是實用世界，
而不是理論世界
```

這個洞見極大地改變了哲學的起點。

---

## 九、哲學從理論走向生活

這裡發生了一個巨大轉向。  
很多古典哲學路線是：

```text
認識
→
世界
```

好像先有理論、  
先有知識、  
先有判斷，  
然後才建立世界理解。

但海德格爾更接近這樣的路線：

```text
生活
→
世界
```

也就是說，  
世界首先不是被認識論建構出來的對象總和，  
而是人已經活在其中、使用其中、受其影響、與之周旋的整體情境。

因此哲學不應從抽離生活的純理論位置出發，  
而應從：

```text
日常存在方式
```

出發。  
這就是海德格爾為什麼會對後來存在主義、詮釋學與現象學  
產生巨大影響。

---

## 十、為什麼這麼重要？

因為海德格爾認為，  
人不是一台理性機器。  
人也不是先有抽象意識、  
再偶爾去面對世界。

人其實是：

```text
始終活在具體情境中的存在
```

我們總是在：

- 某個時間裡
- 某個地方裡
- 某種關係中
- 某種關切下
- 某種實踐任務中

理解世界。  
所以如果哲學忘記這點，  
就會把人誤畫成一個脫離生活的觀察者。  
海德格爾的整個革命，  
就在於把人重新放回世界之中。

---

## 十一、《存在與時間》（Being and Time）

海德格爾最重要的著作是：

```text
Being and Time
《存在與時間》（Sein und Zeit）
```

這本書的核心目標是：

```text
重新提出存在問題
```

他認為，  
如果不知道“存在”是什麼，  
那麼其他哲學問題都缺乏根基。  
因為無論你談真理、科學、主體、自由、歷史，  
都已經預設某種對存在的理解。

所以《存在與時間》不是在普通意義上寫一本“存在學教科書”。  
它更像是在做一件更根本的事情：

```text
把哲學的起點重新安置到存在問題上
```

---

## 十二、存在分析的起點

但存在本身太抽象，  
不能直接憑空定義。  
因此海德格爾提出一個策略：

```text
誰會追問存在？
```

答案是：

```text
人
```

準確地說，  
是那種其存在方式本身會對自己的存在發生關切的存在者。  
因此海德格爾認為：

```text
要理解存在，
就要先分析那個會追問存在的存在者
```

這就引出他下一步的核心概念：

```text
Dasein
此在
```

也就是說，  
他不是先從抽象本體論推演存在，  
而是從人自身的存在方式入手。  
這就是《存在與時間》的分析起點。

## 十三、與過去哲學的比較

如果把你今天這一課放在哲學史裡看，  
可以得到一條很清楚的比較線：

| 哲學家 | 核心問題 |
|---|---|
| 柏拉圖（Plato） | 真實是什麼？ |
| 笛卡兒（René Descartes） | 我如何確定知識？ |
| 康德（Immanuel Kant） | 我如何認識世界？ |
| 胡塞爾（Edmund Husserl） | 意識如何構成世界？ |
| 海德格爾（Martin Heidegger） | 存在是什麼？ |

這不是說海德格爾完全否定前人。  
而是說他認為前面這些路線  
都還沒有真正碰到最根本的問題。  
所以他的工作不是補充一個新理論，  
而是：

```text
把哲學重新拉回存在問題
```

## 十四、一句話總結

如果把今天內容濃縮成一句話，那就是：

```text
海德格爾認為，
整個西方哲學長期過度關注具體存在者與認識問題，
卻遺忘了最根本的存在問題；
哲學真正的任務，
不是先把人當成脫離世界的主體來分析，
而是從人作為「在世存在」的方式出發，
重新追問：什麼叫做存在？
```

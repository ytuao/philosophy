# Day 74｜自由：為什麼自由是一種負擔？

## 今日主題

```text
Day 74｜自由：為什麼自由是一種負擔？
Jean-Paul Sartre
```

前兩天我們學習了薩特（Jean-Paul Sartre）的核心命題：

```text
存在先於本質
```

既然人沒有預設本質，  
那麼接下來最重要的問題就是：

```text
人憑什麼決定自己？
```

薩特的答案是：

```text
自由（Freedom）
```

但他立刻補上一句令人不安的話：

```text
人被判定為自由
Man is condemned to be free
```

今天要理解的，就是為什麼存在主義把自由看成一種沉重處境，  
而不只是美好的權利。

---

## 一、一般人理解的自由

在日常生活裡，  
自由通常代表：

```text
我可以做我想做的事
```

例如：

- 自由旅行
- 自由發言
- 自由選擇職業
- 自由決定生活方式

因此自由常常看起來是一件好事，  
甚至像一種理想狀態。  
在這種理解裡，  
自由主要是：

```text
外在限制比較少
```

這種理解沒有錯，  
但薩特要談的自由  
比這更深，也更讓人不安。

---

## 二、薩特的自由：存在論的自由

薩特談的不是主要意義上的：

- 政治自由
- 法律自由
- 制度保障下的自由權利

他更關心的是：

```text
存在論的自由
```

意思是：

```text
人總是必須在其處境中選擇自己
```

也就是說，  
即使環境有限制，  
你依然得對自己的存在採取某種立場。  
你不能完全不回應，  
也不能完全停止選擇。

所以自由不是附加能力，  
而是人的存在方式本身。  
這正是薩特自由論最嚴厲的地方。

---

## 三、無法逃避選擇

例如你今天對工作不滿意。  
你可能有幾種路：

- 離職
- 忍耐
- 轉職
- 創業
- 改變自己在工作中的投入方式

有人會說：

```text
我沒有選擇
```

薩特會回答：

```text
不，你是在選擇留下
```

這不是說現實限制不存在。  
而是說：

```text
即使你什麼都不做，
那也仍然是一種選擇方式
```

因為“不作為”本身  
也是對處境的某種回應。  
所以自由不是你想不想要的問題，  
而是：

```text
你無法取消自己必須回應處境這件事
```

---

## 四、人被判定為自由

薩特最著名的一句話就是：

```text
人被判定為自由
```

為什麼用“判定”這麼沉重的詞？  
因為：

- 你沒有要求出生
- 你沒有自己挑選起點
- 你不是自己創造這個世界
- 但一旦來到這裡，
  你就必須對自己如何活負責

這種結構裡有一種存在主義特有的荒謬感：  
你不是自願被投進這場遊戲，  
卻必須參與並承擔結果。

所以“被判定”不是誇張說法。  
它恰恰表現出薩特的洞見：

```text
自由不是禮物，
更像一種無法上訴的處境
```

---

## 五、自由帶來焦慮

自由聽起來很好，  
但為什麼很多人其實害怕自由？

因為自由意味著：

```text
沒有保證
```

例如：

- 選擇結婚，沒有人能保證成功
- 選擇創業，沒有人能保證成功
- 選擇移民，也沒有人能保證值得
- 選擇放棄某段關係，沒有人能替你確認那是正確答案

自由真正可怕的地方在於：

```text
你必須自己決定，
而且沒有絕對保證替你背書
```

所以自由不是單純“選項變多”。  
它同時意味著：

```text
不確定性增加，
責任也增加
```

---

## 六、焦慮（Anguish）

薩特延續並改造了海德格爾的思路，  
認為自由會帶來一種特殊情緒：

```text
焦慮（Anguish）
```

這不是一般意義上  
對某件外部事情緊張害怕。  
它更接近：

```text
意識到自己必須決定，
而且沒有最終保證者時的存在震動
```

所以焦慮不是自由的副作用而已。  
它本身正是自由被真正感受到時  
所出現的情緒形式。  
換句話說：

```text
焦慮揭露了自由
```

---

## 七、懸崖的例子

你原稿提到的懸崖例子  
非常能說明薩特的意思。

站在懸崖邊時，  
一般人以為自己害怕的是：

```text
掉下去
```

但薩特會指出，  
更深層的不安在於：

```text
我發現自己其實可以跳下去
```

可怕的不只是外部危險，  
而是突然意識到：

```text
我有能力做出這個行動
```

也就是說，  
真正讓人暈眩的，  
不是崖本身，  
而是自己自由的可能性。  
因此懸崖經驗揭露的不是單純恐懼，  
而是：

```text
自由的眩暈
```

---

## 八、責任的重量

如果自由是真的，  
那麼責任也就是真的。  
因為：

```text
是你在選
```

於是你不能把一切完全推給：

- 命運
- 上帝
- 社會
- 家庭
- 制度
- 過去經驗

當然，  
薩特不是否認環境會影響人。  
更精確地說，  
他承認人總活在某種既定條件中，  
也就是事實性（Facticity）之中。  
但他仍堅持：

```text
你如何理解、承接並回應這些條件，
仍然是你的責任
```

所以責任的重量  
不是來自你能控制一切，  
而是來自：

```text
即使不能控制一切，
你仍然要為自己的回應方式負責
```

---

## 九、自由與價值

自由還帶出另一個更難的問題：

```text
誰決定價值？
```

傳統宗教會說：

```text
上帝決定
```

某些理性主義倫理學會說：

```text
理性決定
```

但薩特認為，  
如果不存在預設本質，  
也不存在先天地寫好的價值表，  
那麼：

```text
價值不是現成等待被發現的東西
```

而是必須透過人的選擇與實踐  
被建立出來。  
因此自由不只是“選手段”，  
還牽涉到：

```text
選擇何者值得
```

---

## 十、創造價值

例如：

- 有人決定家庭最重要
- 有人決定藝術最重要
- 有人決定追求真理最重要
- 有人決定政治正義最重要

對薩特來說，  
這些價值不是像石頭一樣  
被你在世界上撿到。  
更接近的是：

```text
它們在你的選擇與實踐中被賦予分量
```

也就是說，  
價值不是純主觀任性，  
但也不是天上掉下來的絕對命令。  
它是人透過承諾、行動與持續實踐  
使其成為“重要”的。

所以自由的另一層重量就在於：

```text
你不只是在選生活，
你也在選什麼值得被視為有價值
```

---

## 十一、逃避自由

問題是，自由真的很累。  
於是很多人會想逃避。  
典型說法像是：

- 制度逼我的
- 我沒辦法
- 大家都這樣
- 我的個性就是如此
- 我只能這樣活

薩特認為，  
這些說法有時不是單純描述現實，  
而是在進行一種逃避：

```text
逃避自由
```

因為只要我能說  
“不是我選的”，  
我就能減輕責任感。  
而這也正是他提出  
**自欺（Bad Faith）** 的原因。

---

## 十二、自欺（Bad Faith）的本質

自欺不是普通撒謊。  
它更精確地說是：

```text
對自己撒謊
```

也就是說，  
人明明是自由的存在者，  
卻說服自己：

- 我只是制度中的零件
- 我只是某種角色
- 我沒有其他可能
- 我只是被條件推著走

這其實是在把自己  
假裝成一個沒有自由的東西。  
而對薩特來說，  
這是對人之存在最根本的扭曲。  
因為人不是純粹物件，  
而是總要超出當下條件、  
對自身處境採取立場的存在者。

所以自欺的核心就是：

```text
否認自己的自由，
以逃避責任與焦慮
```

---

## 十三、自由與他人

薩特後來也越來越清楚地看到：  
自由並不是孤立的。  
因為世界上不只“我”有自由，  
別人也有自由。  
於是就會出現張力：

```text
我的自由
vs
你的自由
```

例如：

- 我的選擇會影響你
- 你的凝視會限制我對自己的理解
- 我的計畫可能與你的計畫衝突
- 社會生活使自由不再只是個人內在問題

這一點後來會變成  
薩特後期哲學與政治思考的重要方向。  
也就是說，  
自由不只是一個人的冒險，  
它也總是在他人關係中展開。

---

## 十四、存在主義的倫理學

因此存在主義從來不是：

```text
想做什麼就做什麼
```

恰恰相反。  
因為在薩特看來：

```text
每個選擇都在定義你
```

所以自由越大，  
責任越大。  
你不是在隨便選，  
而是在透過選擇決定：

- 你是什麼樣的人
- 你認為什麼重要
- 你願意承擔什麼
- 你接受什麼樣的世界

因此存在主義的倫理張力在於：

```text
既然沒有預設標準，
你就更必須對自己的選擇負責
```

這不是放鬆版倫理學，  
而是高壓版倫理學。

---

## 十五、海德格爾與薩特的差異

這裡也可以看出  
海德格爾與薩特之間的重要差異。

| 海德格爾（Martin Heidegger） | 薩特（Jean-Paul Sartre） |
|---|---|
| 關心存在問題 | 關心自由問題 |
| 強調被拋性與此在分析 | 強調選擇、責任與自我塑造 |
| 焦慮揭露存在 | 焦慮揭露自由 |
| 本真性是承擔存在可能 | 自由是必須為自己選擇並負責 |

海德格爾說：

```text
人被拋入世界
```

薩特則補充說：

```text
即使如此，
人仍然必須選擇
```

所以薩特把存在論進一步推向  
自由與責任的倫理重量。

---

## 十六、一句話總結

如果把今天內容濃縮成一句話，那就是：

```text
薩特認為，
自由不是一種單純令人愉快的權利，
而是人作為存在者無法逃避的處境；
由於人沒有預設本質，也沒有絕對標準替他作決定，
他就必須在既定條件之中自己選擇、創造價值，
並對這些選擇及其後果承擔責任；
正因如此，
自由總與焦慮、自欺的誘惑以及倫理責任緊密相連。
```

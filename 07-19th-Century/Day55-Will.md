# Day 55｜意志：世界真正的驅動力？

## 今日主題

```text
Day 55｜意志：世界真正的驅動力？
Arthur Schopenhauer
```

在昨天的課程中，  
我們已經看到一個非常激進的世界觀：

```text
世界的本質 = 意志（Will）
世界的外在樣貌 = 表象（Representation）
```

但今天要更進一步追問：

```text
這個「意志」到底是什麼？
它只是心理學概念？
還是宇宙級力量？
```

這其實是叔本華（Arthur Schopenhauer）哲學裡最核心的一步。  
因為只要搞錯“意志”的意思，  
整個《作為意志和表象的世界》（The World as Will and Representation）就會被誤讀。

---

## 一、意志不是「選擇能力」

在日常語言裡，  
我們說“有意志”，通常是指：

- 我決定做某件事
- 我能控制自己
- 我有理性選擇能力
- 我能靠意志力克服困難

但叔本華明確說：

```text
這不是他要說的意志
```

在他那裡：

```text
意志 ≠ 理性決策能力
```

而是：

```text
一種更深層、前理性、非概念性的驅動力量
```

也就是說，  
理性選擇只是表層現象。  
真正更深的東西，  
是那股先於理性、推動生命不斷欲求與行動的動力。

---

## 二、意志是生命本身的衝動

叔本華對意志的理解，  
可以簡化為：

```text
想要存在
想要持續
想要延續
想要擴張
```

所以意志首先不是知識問題，  
而是存在問題。

它表現為：

- 生存衝動
- 進食
- 防衛
- 繁殖
- 競爭
- 佔有
- 支配

這些都不是人先做完整理性推演，  
再冷靜決定要不要做。  
相反地，  
這些往往是生命先被推動，  
理性才事後跟上。

因此叔本華的意思是：

```text
生命首先不是「知道」，
而是「想要」
```

---

## 三、意志不只在人類

叔本華的關鍵突破之一是：

```text
意志不是人類專屬
```

如果意志只是人的心理狀態，  
那它就只是一個有限的心理學概念。  
但叔本華不是這樣想。

他認為意志存在於整個自然界的不同層級中，  
例如：

- 植物的生長
- 動物的求生
- 人類的慾望
- 自然界各種力量的持續作用

所以對叔本華而言，  
意志不是個體大腦裡的某種情緒。  
它是：

```text
貫穿世界的根本驅動原理
```

這一步非常重要。  
因為從這裡開始，  
意志就不再是心理學概念，  
而變成：

```text
宇宙論與形上學概念
```

---

## 四、世界是一個「意志階層」

如果把叔本華的想法做一個便於理解的整理，  
可以把世界看成：

```text
意志 → 分層顯現
```

### 第一層：物理世界

- 重力
- 力的作用
- 自然界的推動性

可以被理解為意志最原始、最無意識的表現。

### 第二層：生命世界

- 生長
- 繁殖
- 適應
- 自我維持

這裡的意志已經顯現為明顯的生命衝動。

### 第三層：意識世界

- 欲望
- 恐懼
- 焦慮
- 痛苦
- 追求

到了人類這一層，  
意志不只活動，  
還會被感受到。  
也就是說，  
人第一次在自身內部經驗到：

```text
自己是被欲求推動的存在
```

---

## 五、理性只是工具，不是主人

叔本華極力反對理性中心主義。  
他認為：

```text
理性不是主人，
而是意志的工具
```

例如：

```text
你想吃東西（意志）
你計算怎麼取得食物（理性）
```

再例如：

```text
你想獲得地位（意志）
你設計路線、建立策略、說服別人（理性）
```

所以理性的角色是：

- 計算
- 規劃
- 比較
- 預測
- 找方法

但理性通常不決定“最根本想要什麼”。  
它只是服務於那個更深的“想要”。

因此叔本華幾乎把理性降格為：

```text
意志的僕人
```

這與黑格爾（Georg Wilhelm Friedrich Hegel）  
把理性看成世界本質的立場，  
形成最尖銳的對立。

---

## 六、為什麼意志是「盲目的」？

叔本華一直強調：

```text
意志是盲目的
```

這裡的“盲目”不是說它完全沒有作用，  
而是說它：

- 不以真理為目的
- 不以幸福為目的
- 不以理性自洽為目的
- 不以終局圓滿為目的

它真正做的只有一件事：

```text
繼續欲求
繼續推動
繼續存在
```

所以它之所以盲目，  
是因為它沒有最後的明智目標。  
它只是無休止地要求：

```text
更多生命
更多延續
更多佔有
更多存在
```

這也是叔本華為什麼認為  
意志永遠不會真正滿足。  
因為它本性上就不是為了安息而存在。

---

## 七、意志 = 無終點的運動

因此意志的結構可以寫成：

```text
欲望
→ 行動
→ 暫時滿足
→ 新欲望
→ 無限循環
```

這裡最重要的是：

```text
沒有終點
```

因為意志本身不是在朝一個最終完成狀態前進。  
它不是黑格爾式的自我完成歷史。  
它沒有最高和解，  
沒有終極圓滿，  
也沒有理性終點。

所以叔本華看到的世界不是：

```text
發展
```

而是：

```text
反覆欲求
```

也就是說，  
生命的本性不是完成，  
而是持續追逐。

---

## 八、生命為什麼痛苦？

叔本華的回答非常簡潔，  
也非常冷酷：

```text
因為意志永遠超過現實
```

也就是：

```text
想要 > 擁有
```

只要這個結構存在，  
生命就不可能安穩。

因此人生會落入這樣的循環：

```text
缺乏 → 痛苦
滿足 → 無聊
再次欲望 → 新痛苦
```

如果沒有得到，  
人會痛苦。  
如果得到了，  
那種緊張解除後又會陷入空虛。  
而空虛很快又催生新的欲望。

所以痛苦不是偶發事件，  
而是生命結構的必然產物。

---

## 九、意志不是「個人」，而是宇宙結構

叔本華最關鍵的轉變之一是：

```text
不是「我有意志」
而是「意志透過我而顯現」
```

這句話非常重要。

在日常理解裡，  
我們把意志當成“我的一種能力”。  
但叔本華反過來說：

```text
個體不是意志的主人，
個體只是意志的局部表現
```

也就是說，  
“我”不是根本實體。  
更根本的其實是那個普遍的意志，  
而個體只是它在現象界的一個暫時形態。

這使叔本華的哲學  
帶有很強的去個體中心化色彩。

---

## 十、個體只是「現象」

如果延續上面的思路，  
就會得到一個更激進的結論：

```text
個體並不是最終實在
```

因為個體屬於：

```text
表象界
現象界
```

它是在時間、空間、因果性之中  
被區分出來的存在。  
但在更深的本體層面，  
真正存在的是：

```text
統一的意志
```

因此不同個體之間的分離，  
在叔本華看來，  
更像是現象層面的分別，  
而不是最終本質上的割裂。

這一步對他後面的倫理學非常關鍵。

---

## 十一、這導致什麼結論？

如果叔本華是對的，  
那麼會出現一個很強的結論：

```text
個體之間沒有最終本質差異
```

因為：

- 所有人類
- 所有動物
- 所有生命
- 甚至整個自然界

都只是：

```text
同一意志的不同顯現
```

這表示我們平常強烈感受到的  
“我”和“他者”的界線，  
在最深層本體上不再那麼絕對。

而這也說明了為什麼叔本華後來會走向：

```text
同情（Compassion）
```

作為道德的重要基礎。

---

## 十二、道德的基礎（延伸）

雖然今天主題不是倫理學，  
但這裡其實已經自然通向叔本華的倫理結論。

如果所有生命都只是  
同一意志的不同表現，  
那麼：

```text
傷害他人，
不只是傷害一個外部對象；
更是在同一存在根基中的內部衝突。
```

因此叔本華的倫理學不是建立在純理性義務上，  
而更傾向建立在：

```text
同情（Compassion / Mitleid）
```

之上。

因為當人真正看見  
他者與自己在本體上的相通，  
自我中心的界線就會鬆動。  
而道德也就可能成立。

---

## 十三、與黑格爾的根本對立

叔本華與黑格爾的對立，  
到這裡已經非常清楚。

| 黑格爾（Georg Wilhelm Friedrich Hegel） | 叔本華（Arthur Schopenhauer） |
|---|---|
| 理性是世界本質 | 意志是世界本質 |
| 歷史通向自由 | 歷史無法消除痛苦 |
| 發展是自我完成 | 生命是反覆欲求 |
| 矛盾可被揚棄 | 欲望不斷再生 |
| 世界有理性方向 | 世界由盲目衝動推動 |

所以從黑格爾到叔本華，  
哲學史上發生了一次極劇烈的翻轉：

```text
理性宇宙
→ 非理性宇宙
```

---

## 十四、哲學史的巨大斷裂

這個轉變之所以重要，  
是因為它徹底改變了“世界是什麼”這個問題的答案。

在黑格爾那裡，  
世界仍然可以被理解為：

- 可理解的
- 有方向的
- 有歷史意義的
- 能被理性掌握的

但在叔本華這裡，  
世界更像是：

- 盲目的
- 無目的的
- 無終點的
- 由深層驅力推動的

因此叔本華的重要性，  
不只在於他提出“意志”這個概念。  
更在於他正式把西方哲學  
從理性中心拉向：

```text
非理性
欲望
驅力
深層生命衝動
```

這條路後來會深刻影響尼采（Friedrich Nietzsche）  
與佛洛伊德（Sigmund Freud）等人。[web:217][web:233]

---

## 十五、一句話總結

如果把今天的內容濃縮成一句話，那就是：

```text
叔本華認為，
意志不是心理上的選擇能力，
而是貫穿宇宙、前理性、盲目且永不滿足的生命衝動；
它構成世界的本體，
並驅動一切存在陷入無止境的欲求循環。
```

# Day 27｜阿奎那：理性如何證明神存在？

## 今日主題

```text
Day 27｜阿奎那（Thomas Aquinas）
理性如何證明神存在？
```

今天要學習整個中世紀哲學的巔峰人物之一：

```text
Thomas Aquinas（阿奎那）
```

如果說：

- 奧古斯丁代表「柏拉圖化的基督教」
- 安瑟倫代表「理性分析信仰」

那麼阿奎那代表的就是：

```text
亞里士多德
+
基督教
=
中世紀哲學高峰
```

他的思想深刻影響：

- 天主教神學
- 西方自然法理論
- 現代政治哲學
- 現代倫理學

直到今天，仍然是極其重要的思想來源。

## 一、阿奎那是誰？

阿奎那約出生於 1225 年，生活在中世紀盛期。  
這個時代發生了一件極其重要的大事：

```text
大量亞里士多德著作重新被翻譯進入歐洲
```

在此之前，西方思想長期主要受：

- 柏拉圖
- 奧古斯丁
- 新柏拉圖主義

的影響。

但隨著亞里士多德的形而上學、物理學、倫理學、邏輯學重新進入學術世界，  
一個重大問題出現了：

```text
如果亞里士多德說得對，
那麼基督教還能成立嗎？
```

阿奎那的偉大之處就在於：  
他不是逃避這個問題，而是正面迎戰。

---

## 二、理性與信仰會衝突嗎？

這是整個中世紀哲學最核心的問題之一。

有些人認為：

```text
信仰
高於
理性
```

有些人則認為：

```text
理性
高於
信仰
```

而阿奎那提出第三條路：

```text
理性
與
信仰
來自同一真理
```

因此，他的核心立場是：

```text
真正的理性
不會反對真正的信仰
```

原因很簡單。  
如果真理只有一個來源，而神又是真理的根源，  
那麼來自理性的真理與來自啟示的真理，最終不可能真正矛盾。  
若看起來矛盾，要麼是理性推理出錯，要麼是對啟示理解錯了。

---

## 三、阿奎那反對安瑟倫

昨天與前天學過安瑟倫的本體論證。  
安瑟倫主張：

```text
從神的概念
推出神存在
```

但阿奎那不同意。

他的理由是：

```text
人類並不知道神的本質
```

如果我們根本無法直接把握神是什麼，  
那麼就不能單憑對“神”的概念分析，推出神存在。

因此，阿奎那認為：

```text
不能從概念出發，
必須從世界出發。
```

也就是說，  
我們雖然不能直接看見神的本質，  
但可以從世界中的現象、因果、秩序與存在方式，  
一步步理性地推向神。

這就是他與安瑟倫最大的分歧：

- 安瑟倫：從概念到神
- 阿奎那：從世界到神

---

## 四、五路證明

阿奎那最著名的思想之一，就是：

```text
Five Ways（五路證明）
```

他認為，  
只要仔細觀察世界，  
理性就會一步步指向一個最終根源。  
這五條路不是五個完全孤立的證明，  
而是五種不同角度的理性推進方式。

---

## 五、第一條路：運動論證

第一條路是：

```text
Motion（運動論證）
```

阿奎那觀察到，世界中的萬物都在變化。  
例如：

- 木頭燃燒
- 樹木生長
- 行星運行
- 水被加熱
- 種子變成植物

在亞里士多德式語境中，這裡的“運動”不只是位置移動，  
而是廣義上的：

```text
從潛能到現實的變化
```

例如，一塊冷的鐵本來“有可能變熱”，  
被火加熱後，這種可能性變成了現實。

阿奎那接著說：

```text
凡是被推動者，
都由別的東西推動。
```

於是形成鏈條：

```text
A 推動 B
B 推動 C
C 推動 D
```

但這條鏈不能無限倒退。  
因為如果沒有第一推動者，  
後面的一切推動都不會開始。

所以，他得出結論：

```text
必須有第一推動者
```

而阿奎那稱之為：

```text
神
```

---

## 六、第二條路：因果論證

第二條路是：

```text
Efficient Cause（因果論證）
```

世界中的事物都有原因。  
例如：

- 房屋 ← 建築工人
- 工人 ← 父母
- 父母 ← 更早之前的原因鏈

沒有什麼東西是自己的有效因。  
因為如果某物是自己的原因，  
那它就必須在自己存在之前先存在，這顯然不可能。

所以，世界中的事物處在因果鏈條中。  
但阿奎那認為，這條因果鏈也不能無限向後延伸。  
因為如果沒有第一原因，  
那麼中間原因和最終結果也都不會存在。

因此必須有：

```text
第一原因
```

而這第一原因就是：

```text
神
```

要注意，阿奎那在這裡關心的不是“很久以前宇宙最早有個開端”，  
而是“現在這一整套因果秩序之所以成立，必須依賴一個最終根據”。

---

## 七、第三條路：偶然與必然

第三條路是：

```text
Contingency（偶然與必然）
```

阿奎那觀察到，世界中的大部分事物都屬於：

```text
偶然存在
```

也就是說，它們：

```text
可能存在，
也可能不存在。
```

例如：

- 人類
- 動物
- 國家
- 行星
- 樹木

它們都不是非存在不可的。  
它們有開始，也可能消失。

阿奎那接著提出一個推論：  
如果一切都只是偶然存在，  
那麼理論上曾經可能有一刻：

```text
什麼都不存在
```

但如果真有那樣一刻，  
那麼現在也不會有任何東西存在，  
因為：

```text
無法從無生有
```

但現在確實有東西存在。  
因此，不可能一切都只是偶然存在。  
必須有一個本身不是偶然的，而是：

```text
必然存在者
```

這個存在者不是從別的東西獲得存在，  
而是自身就具有存在的必然性。  
阿奎那把它稱為：

```text
神
```

---

## 八、第四條路：完美程度

第四條路是：

```text
Degrees of Perfection（完美程度）
```

我們日常常會說：

- 這個更善
- 那個更真
- 這個更美
- 那個更高貴

也就是說，  
我們經常在比較事物的程度差異。

阿奎那認為：

```text
如果有程度比較，
就必須有最高標準。
```

也就是說，  
若我們能說某物更善，  
那麼就預設了某種“最善”的標準；  
若我們能說某物更真，  
那麼就預設了某種“最真”的尺度。

因此他推論：

```text
最高善
最高真
最高美
必須存在
```

而這個最高、最圓滿、最完美的存在，  
阿奎那稱之為：

```text
神
```

這條路是五路中最容易引發爭議的一條，  
因為很多現代人會質疑：  
“有比較”是否真的需要一個“最高實體”作為根據？

---

## 九、第五條路：目的論證

第五條路是：

```text
Teleological Argument（目的論證）
```

阿奎那觀察到，  
自然界中的許多事物似乎都朝向某種目的運作。  
例如：

- 眼睛適合看
- 種子適合發芽
- 生物器官適合特定功能
- 天體運動具有規律
- 自然過程呈現秩序

阿奎那認為，  
即使許多自然事物本身沒有意識，  
它們卻仍然穩定地朝向某種結果。  
這表示它們似乎被導向某種目的。

於是他提出類比：

```text
箭會射向目標，
因為有射手引導。
```

同樣地：

```text
自然之所以具有目的性，
也需要某個理智存在加以引導。
```

因此，他得出結論：

```text
宇宙設計者
=
神
```

這條路後來對設計論、自然神學以及近代許多神存在證明影響非常深。

---

## 十、五路證明整理

可以把五路證明簡單整理如下：

| 路徑 | 起點 | 結論 |
|---|---|---|
| 運動 | 變化存在 | 第一推動者 |
| 因果 | 因果存在 | 第一原因 |
| 偶然 | 事物會消失 | 必然存在者 |
| 完美 | 有程度差異 | 最高完美 |
| 目的 | 自然秩序 | 設計者 |

這五條路共同的特點是：

```text
從世界出發
```

它們都不是先分析“神”這個詞，  
而是先看現實中的：

- 變化
- 因果
- 存在方式
- 價值程度
- 目的秩序

再一步步推向最終根源。

---

## 十一、與安瑟倫比較

阿奎那和安瑟倫的差別，可以非常清楚地整理如下：

| 安瑟倫 | 阿奎那 |
|---|---|
| 先驗 | 經驗起點 |
| 從概念出發 | 從世界出發 |
| 本體論證 | 五路證明 |
| 偏向概念分析 | 偏向自然神學 |

可以再濃縮成一句話：

```text
安瑟倫：
概念
↓
神

阿奎那：
世界
↓
神
```

這個差異非常重要。  
它不只是兩個神存在證明的不同，  
而是兩種理性使用方式的不同。

---

## 十二、阿奎那的知識理論

阿奎那的知識理論深受亞里士多德影響。  
他認為：

```text
人類知識來自：
感官
↓
理性
↓
知識
```

也就是說，人不是一開始就直接擁有現成真理，  
而是先透過感官接觸世界，  
再由理性整理、抽象、分析，形成知識。

這和奧古斯丁那種偏重內在光照的路線相比，  
更強調經驗起點。  
也更接近亞里士多德的經驗主義精神。

---

## 十三、信仰與理性的分工

阿奎那並不認為理性能知道一切。  
他主張的是：

```text
有些事情理性能知道，
有些事情只能靠信仰知道。
```

例如，理性可以嘗試證明：

- 神存在
- 世界有秩序
- 某些道德原則具有普遍性

但有些內容則只能依靠啟示，例如：

- 三位一體
- 道成肉身
- 某些救恩真理

因此阿奎那的立場不是“理性萬能”，  
而是：

```text
理性有其尊嚴，
但也有其界限。
```

這種分工思想，是中世紀最成熟的理性與信仰協調方案之一。

---

## 十四、後世批評

後來很多哲學家都對五路證明提出質疑。  
例如：

```text
David Hume（休謨）
```

會質疑以下幾點：

- 因果鏈未必需要第一原因
- 宇宙未必像人工製品那樣需要設計者
- 我們從局部秩序推到整體設計者，是否過度推論
- 就算真有第一因，也未必等於基督教意義上的神

現代科學也會對阿奎那提出挑戰。  
例如，進化論會讓第五路的“目的性”解釋變得更複雜；  
宇宙學則會重新打開“宇宙是否必須有第一因”的問題。

所以今天即使很多人不接受五路證明，  
也不代表它們沒有價值。  
因為它們提出的問題，至今仍然沒有被完全消化。

---

## 十五、哲學史地位

即使今天不接受五路證明，  
阿奎那仍然是哲學史上的巨人。

因為他完成了一件極其重要的事：

```text
亞里士多德
+
基督教
的深度整合
```

更重要的是，  
他建立了：

```text
信仰與理性合作
```

的典範。

在他這裡，理性不是信仰的敵人，  
也不是信仰的主人，  
而是信仰的重要盟友。

這種結構後來深刻影響：

- 神學
- 法哲學
- 自然法理論
- 倫理學
- 政治哲學

阿奎那的影響之所以長久，  
不只是因為他提供答案，  
更是因為他示範了一種思想方法：  
如何把不同傳統整合成系統。

---

## 十六、一句話總結

如果把阿奎那的核心思想濃縮成一句話，那就是：

```text
只要仔細觀察世界，
理性就能一步步走向神。
```

這句話代表了他和安瑟倫最根本的差異。  
安瑟倫相信概念足以推向存在，  
而阿奎那相信：  
人類必須從世界出發，  
才能理性地走向超越世界的根源。

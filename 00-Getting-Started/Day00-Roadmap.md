### 第一階段：哲學的誕生（第 1–3 週）

```text
核心問題
- 世界到底是什麼？
- 人應該怎樣活？
```

**學習對象**

- 前蘇格拉底哲學家（Pre-Socratics）
- 蘇格拉底（Socrates）
- 柏拉圖（Plato）
- 亞里斯多德（Aristotle）

**這一階段要理解的內容**

- 理性（reason / logos）為什麼誕生
- 哲學為何脫離神話（myth）
- 西方思想的根基是什麼

**重點概念**

- Logos（理性 / reason）
- 德性（virtue）
- 靈魂（soul）
- 形式／理念（form / idea）
- 本體（being）

### 第二階段：宗教與理性（第 4–6 週）

```text
核心問題
- 信仰（faith）與理性（reason）能否共存？
```

**學習對象**

- 斯多噶學派（Stoicism / Stoics）
- 伊比鳩魯（Epicurus）
- 基督教哲學（Christian philosophy）
- 奧古斯丁（Augustine）
- 托馬斯．阿奎那（Thomas Aquinas）

**這一階段要理解的內容**

- 為什麼歐洲進入「神學時代」
- 自由意志（free will）問題
- 上帝證明（proofs of God）

### 第三階段：現代哲學革命（第 7–12 週）

這是整體路線中最重要的階段。

```text
核心問題
- 人類如何獲得知識？
```

**學習對象**

- 笛卡兒（René Descartes）
- 斯賓諾莎（Baruch Spinoza）
- 萊布尼茲（Gottfried Wilhelm Leibniz）
- 洛克（John Locke）
- 柏克萊（George Berkeley）
- 休謨（David Hume）
- 康德（Immanuel Kant）

**核心主題**

- 認識論革命（epistemological revolution）
- 懷疑主義（skepticism）
- 經驗主義（empiricism）vs 理性主義（rationalism）
- 主體（subject）
- 因果性（causality）
- 先驗條件（a priori conditions / transcendental conditions）

**關鍵人物**

- 康德（Immanuel Kant）是關鍵中的關鍵

### 第四階段：19 世紀爆炸（第 13–18 週）

```text
核心問題
- 歷史是什麼？
- 意志（will）是什麼？
- 自由（freedom）是什麼？
- 人類意義（meaning）是什麼？
```

**學習對象**

- 黑格爾（G. W. F. Hegel）
- 叔本華（Arthur Schopenhauer）
- 齊克果（Søren Kierkegaard）
- 馬克思（Karl Marx）
- 尼采（Friedrich Nietzsche）

**這一階段要看到的內容**

- 現代社會批判（critique of modern society）
- 虛無主義（nihilism）
- 辯證法（dialectic）
- 存在焦慮（existential anxiety）

### 第五階段：20 世紀哲學分裂（第 19–24 週）

20 世紀哲學大致分成兩大傳統：歐陸哲學（Continental Philosophy）與分析哲學（Analytic Philosophy）。

#### 歐陸哲學（Continental Philosophy）

```text
核心問題
- 人如何存在？
```

**學習對象**

- 胡塞爾（Edmund Husserl）
- 海德格爾（Martin Heidegger）
- 薩特（Jean-Paul Sartre）
- 傅柯（Michel Foucault）
- 德希達（Jacques Derrida）

#### 分析哲學（Analytic Philosophy）

```text
核心問題
- 語言如何構成世界？
```

**學習對象**

- 弗雷格（Gottlob Frege）
- 羅素（Bertrand Russell）
- 維特根斯坦（Ludwig Wittgenstein）
- 奎因（W. V. O. Quine）

### 第六階段：21 世紀議題（最後 2 週）

```text
現代主題
- AI 倫理（AI ethics）
- 意識問題（problem of consciousness）
- 後人類（posthumanism）
- 性別哲學（philosophy of gender）
- 政治正確（political correctness）
- 技術統治（technocracy / technological governance）
- 生態哲學（environmental philosophy / ecological philosophy）
```

**最終理解**

- 哲學不是歷史，而是今天仍在發生的戰爭

## 學習哲學的正確方法

不要先問：

```text
「這是真的嗎？」
```

要先問：

```text
「他為什麼會這樣想？」
```

哲學史的本質是：

```text
一個問題，被不斷重新回答。
```

### 問題鏈示例：世界是什麼？

```text
Thales（泰勒斯）            → 水（water）
Heraclitus（赫拉克利特）   → 變化（change）
Plato（柏拉圖）           → 理念（Forms / Ideas）
Aristotle（亞里斯多德）    → 實體（substance）
Descartes（笛卡兒）        → 精神與物質（mind and matter）
Kant（康德）                → 現象與物自體（phenomena and noumena）
Heidegger（海德格爾）         → 存在（Being）
```

這代表學習哲學時，真正要追蹤的不是人物名單本身，而是問題如何演化。

# Day 50｜謝林：自然也是精神嗎？

## 今日主題

```text
Day 50｜謝林：自然也是精神嗎？
Naturphilosophie（自然哲學）
```

在費希特（Johann Gottlieb Fichte）之後，  
德國觀念論（German Idealism）立刻遇到一個關鍵問題：

```text
如果一切都從自我出發，
那自然到底是什麼？
```

費希特把世界大幅收進“自我”的活動之中。  
但問題也立刻浮現：

```text
如果自然只是「我設立的非我」，
那自然似乎就失去了自己的獨立性
```

這讓許多人感到不滿。  
於是下一位重要哲學家登場：

```text
謝林（Friedrich Wilhelm Joseph Schelling）
```

---

## 一、謝林的核心反轉

謝林的出發點非常關鍵。

他認為：

```text
自然不能只是被主體設立出來的東西
```

如果自然只是自我的對立物，  
那麼自然就變得太被動、太附屬。  
這無法解釋生命、生成、組織性與自然自身的活力。

因此謝林提出一個極著名的思路：

```text
自然本身也是一種精神活動
```

後人常用一句話來概括：

```text
自然 = 可見的精神
精神 = 不可見的自然
```

這句話的真正意思不是把石頭說成會思考，  
而是說：

```text
自然與精神不是彼此絕對分裂的兩個領域，
而是同一實在的不同顯現方式
```

---

## 二、費希特 vs 謝林

兩人的差異可以簡單對照：

### 費希特（Johann Gottlieb Fichte）

```text
自我
→ 創造世界
→ 自然只是對立物
```

### 謝林（Friedrich Wilhelm Joseph Schelling）

```text
自然 ≠ 被創造的附屬物
自然 = 與精神同源
```

也就是說，  
費希特比較偏向：

```text
主體中心
```

而謝林則試圖走向：

```text
整體中心
```

他不願意讓自然完全依附於主體。  
他要說的是：

```text
自然本身就有自己的生產性與展開力量
```

---

## 三、核心概念：同一性哲學

謝林後來發展出一個重要方向，  
常被稱為：

```text
Identity Philosophy
同一性哲學
```

它的基本意思是：

```text
自然與精神不是兩個根本不同的東西，
而是同一個實在的兩種表現
```

也就是說，  
在最深層次上：

- 主體與客體
- 精神與自然
- 思維與存在

並不是絕對對立的。  
它們只是同一根本實在  
在不同層面上的顯現。

因此謝林想做的事情，  
不是讓精神吞掉自然，  
也不是讓自然消滅精神，  
而是找到兩者的共同根源。

---

## 四、自然不再是「死物」

在很多傳統觀點中，  
尤其在機械自然觀影響下，  
自然常被理解成：

- 被動物質
- 機械系統
- 因果運作的死結構
- 沒有內在目的性的東西

謝林強烈反對這種看法。

他認為：

```text
自然不是死物，
而是活的過程
```

自然不是單純的機械堆積，  
而是一個：

```text
自我展開、自我組織、自我提升的整體
```

這樣一來，  
自然就不只是可被外在測量的對象，  
而是具有內在生產性的現實。

---

## 五、自然的「內在精神性」

謝林認為，  
自然本身具有某種內在方向性。

例如：

- 生命的生成
- 有機體的形成
- 結構的複雜化
- 自然由低級向高級層次的展開

在他看來，  
這些不應只被理解成偶然組合。  
它們顯示出：

```text
自然內部有某種理性化的運動
```

這裡的“理性”不是說自然像人一樣坐下來思考，  
而是說：

```text
自然不是純粹混亂，
而是具有可展開、可生成、可提升的秩序
```

所以謝林試圖賦予自然一種內在主動性。  
這正是他和費希特的重要差別。

---

## 六、自然 = 無意識精神

謝林有一個非常有名的想法：

```text
自然是無意識的精神
```

而：

```text
精神是自然達到自我意識的階段
```

這句話非常關鍵。  
它意味著世界不是分成兩塊：

- 一邊是自然
- 一邊是精神

而是一條連續鏈條：

```text
自然
→ 生命
→ 意識
→ 理性
→ 自我意識
```

所以人類精神不是突然從天外掉進自然界的東西。  
它是自然發展到某個高度之後，  
在自身中達到自我認識的結果。

這讓謝林的世界觀具有很強的整體性與生成性。

---

## 七、哲學的重大轉向

從費希特到謝林，  
哲學的重心再次發生改變。

### 費希特

```text
世界 = 自我活動的產物
```

### 謝林

```text
自我 + 自然 = 同一實在的不同層面
```

也就是說：

```text
不是自我創造自然，
而是自然本身已經具有精神性
```

這一步非常重要。  
因為它把自然重新放回哲學中心。  
不再只是主體哲學，  
而變成一種更整體的存在論嘗試。

---

## 八、自然哲學（Naturphilosophie）

謝林建立的：

```text
Naturphilosophie
自然哲學
```

其核心思想可以概括為：

```text
自然不是死的物質集合，
而是具有內在生命與發展方向的整體
```

這種自然哲學和單純自然科學不同。  
它不是只做實驗、分類、測量。  
它想追問的是：

```text
自然作為整體，
其內在原理與生成結構是什麼？
```

也就是說，  
謝林並不滿足於“自然如何運作”這種問題，  
他還要問：

```text
自然何以能生成生命、意識與精神？
```

---

## 九、自然中的「對立運動」

謝林認為，  
自然的運作方式不是靜止的，  
而是透過對立與張力展開。

可以簡化成：

```text
對立
→ 張力
→ 統一
→ 更高層次
```

例如：

- 冷與熱
- 吸引與排斥
- 有機與無機
- 主動與被動

這些不是偶然拼在一起的元素。  
在謝林看來，  
它們表現出自然內在的動力結構。

雖然謝林的“辯證”還不像後來黑格爾（Georg Wilhelm Friedrich Hegel）那麼系統化，  
但已經有很明顯的方向：

```text
自然透過對立運動推進自身
```

---

## 十、為什麼這很重要？

這種看法會直接改變我們對“自然”的理解。

### 傳統機械自然觀

```text
自然 = 可測量的機械系統
```

### 謝林的自然觀

```text
自然 = 活的、發展中的、整體性的實在
```

這一點非常重要。  
因為它影響的不只是哲學，  
還包括：

- 浪漫主義（Romanticism）
- 19 世紀生物學想像
- 有機體觀
- 黑格爾哲學
- 後來對機械論的批判

所以謝林的自然哲學，  
雖然今天看起來不屬於嚴格現代科學，  
但它在思想史上極具啟發性。

---

## 十一、藝術與自然的關係

謝林不只關心自然與精神，  
也非常重視：

```text
藝術（Art）
```

他認為藝術具有特殊地位。  
因為在藝術中：

- 主體性不是純抽象理性
- 客觀性也不是死物機械
- 自由與必然似乎達到某種統一

因此在謝林看來：

```text
藝術是自然與精神統一的高級顯現
```

或者也可以說：

```text
藝術讓無意識的自然力量
與有意識的精神創造
在作品中相遇
```

所以藝術不是附屬品。  
在德國觀念論脈絡中，  
藝術常常被看成理解絕對者的重要途徑。

---

## 十二、自然與自由的統一

謝林其實在試圖解決一個非常難的問題：

```text
自由（主體）
vs
自然（客體）
```

傳統上，  
兩者經常被看成對立：

- 自然是必然的
- 精神是自由的

但謝林的答案是：

```text
兩者本來就是同一實在的不同層面
```

也就是說，  
自由不是從自然外面插進來的。  
自然也不是自由的敵人。  
相反地：

```text
自由是自然在更高階段上達到自我顯現的形式
```

這種想法非常雄心勃勃。  
它想要消解自然與精神、  
必然與自由之間的根本裂縫。

---

## 十三、從費希特到謝林的進展

可以把兩人的發展簡化成這樣：

```text
費希特（Johann Gottlieb Fichte）
世界 = 自我設立

↓
謝林（Friedrich Wilhelm Joseph Schelling）
世界 = 自然與精神的同一體
```

這表示德國觀念論正在一步步擴大自己的視野：

- 從康德的主體條件
- 到費希特的自我活動
- 再到謝林的自然—精神同源論

哲學不再只問“我如何認識”，  
而是開始問：

```text
整個實在如何以自然與精神的形式展開？
```

這正是在為黑格爾鋪路。

---

## 十四、但問題來了

謝林的理論雖然迷人，  
也立刻引發不少質疑。

### 問題 1：如何證明？

如果你說：

```text
自然是精神
```

那麼：

```text
怎麼證明？
怎麼驗證？
```

這會讓很多人懷疑，  
謝林是否已經離開可論證哲學，  
走向過度宏大的形上學敘事。

### 問題 2：是否過於詩意？

很多批評者認為，  
謝林的語言非常有魅力，  
但有時更像：

```text
詩
```

而不像嚴格意義上的哲學論證或科學理論。

### 問題 3：是否重新回到不可驗證形上學？

康德（Immanuel Kant）辛苦劃定理性邊界，  
但謝林似乎又重新引入一種很難驗證的整體性本體論。

這就讓人懷疑：

```text
他是不是重新打開了
康德原本想關上的那扇門？
```

---

## 十五、哲學史的下一步

謝林的思想將直接引向下一位更具系統性的哲學家：

```text
黑格爾（Georg Wilhelm Friedrich Hegel）
```

黑格爾將嘗試把：

- 費希特的自我活動
- 謝林的自然—精神同一性

整合進一個更宏大的體系：

```text
歷史性的整體運動
```

也就是說，  
黑格爾會把問題進一步推向：

```text
不是單純自我，
也不是單純自然，
而是絕對精神（Absolute Spirit）
如何在歷史中展開自身
```

所以謝林在哲學史上的地位非常關鍵。  
他是一座橋：

```text
從主體哲學
通向整體存在論與歷史哲學
```

---

## 十六、一句話總結

如果把謝林的思想濃縮成一句話，那就是：

```text
自然不是被動的物質，
而是與精神同源的活的過程；
自然與精神本質上是同一實在的不同表現。
```
